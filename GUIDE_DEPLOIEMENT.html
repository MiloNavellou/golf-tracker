🚀 Mettre l'app en ligne

Option simple — Netlify Drop (aucune installation)

1 Extrais le dossier golf_pwa de ce ZIP sur ton PC
2 Va sur app.netlify.com/drop
3 Glisse le dossier golf_pwa dans la zone → URL en ligne en 30 secondes 🎉

Option Vercel (ligne de commande)

1 Installe Node.js depuis nodejs.org
2 Ouvre un terminal dans le dossier golf_pwa et tape : 
npm install -g vercel
vercel login
vercel
Laisse les valeurs par défaut. Vercel te donne une URL https://...vercel.app 🎉
💡 Les deux options donnent une URL HTTPS, indispensable pour que la PWA s'installe et que la connexion fonctionne.

🔐 Première connexion

1 Ouvre l'URL de ton app
2 Clique "Créer un compte", entre email + mot de passe (6 caractères min)
3 Connecté immédiatement. Mêmes identifiants sur tes autres appareils = tu retrouves tes parties.

📱 Installer sur iPhone

1 Ouvre l'URL dans Safari
2 Bouton Partager ↑ → "Sur l'écran d'accueil"
3 Icône ⛳ ajoutée. Connecte-toi une fois, c'est gardé.

💻 Installer sur PC

1 Ouvre l'URL dans Chrome ou Edge
2 Icône ⊕ Installer dans la barre d'adresse

🔒 Tes données

Chaque partie est rattachée à ton compte. La règle de sécurité Supabase (RLS) fait que personne d'autre ne peut voir tes parties, même avec l'URL. La partie en cours non terminée est gardée localement jusqu'à ce que tu la termines, puis elle part dans le cloud.

⚠️ Une partie "en cours" non terminée reste sur l'appareil où tu l'as commencée. Termine-la pour qu'elle soit visible partout.
